O K-TP Projeto TRANSFORMAÇÃO é um programa abrangente que atende a todas as faixas etárias, proporcionando resultados efetivos para pessoas de diferentes idades.

O fundador Say Korvil, profissional de ed. física, treinador bodybuilder, terapeuta holístico, lidera o projeto com sua experiência de 16 anos no campo do fitness, saúde e desenvolvimento pessoal.

Para o objetivo individual de cada participante, o estado atual precisa estar alinhado com as etapas do projeto. As etapas serão ajustadas para atender o objetivo específico, garantindo resultados efetivos.
